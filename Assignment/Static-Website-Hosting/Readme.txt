## üìÅ How to Deploy

1. **Create an S3 bucket** with static website hosting enabled.
2. **Upload all files** from the `Static-Website-Hosting` folder to the S3 bucket.
3. Set `index.html` as the index document in the S3 static website settings.
4. Make the files public or set a bucket policy for public read access.
5. Access your website using the S3 website endpoint.
